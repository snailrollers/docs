## Obrazovky displeje

Spooler má OLED displej 128×64 px s několika různými obrazovkami. Mezi nimi přepínáte krátkým stiskem tlačítka.

### Hlavní obrazovka — Měření

```
┌──────────────────────────────┐
│ Spooler          🔋 82%      │
│                              │
│     1 247.3 m                │
│                              │
│ ████████████░░░░  Cívka A1   │
│ 750 m zbývá                  │
└──────────────────────────────┘
```

| Prvek | Popis |
|-------|-------|
| Velké číslo | Aktuální naměřená délka od posledního resetu |
| Progress bar | Vizualizace spotřeby cívky (zaplnění = spotřeba) |
| Zbývá | Zbývající délka kabelu na cívce |
| Baterie | Stav baterie v % |
| Jméno cívky | Název přiřazené cívky (z mobilní aplikace) |

## Stavové ikony

| Ikona | Stav | Popis |
|-------|------|-------|
| 🔋 zelená | >50% | Baterie v pořádku |
| 🔋 žlutá | 20–50% | Baterie se vybíjí |
| 🔋 červená | <20% | Nízká baterie |
| ⚡ | nabíjení | USB-C připojeno |
| 📶 | BLE aktivní | Zařízení inzeruje |
| 🔗 | BLE připojeno | Aktivní BLE spojení |
| ⚠️ | varování | Chyba enkodéru nebo NVS |

## Ovládání tlačítkem

| Akce | Funkce |
|------|--------|
| Krátký stisk (< 1s) | Přepnutí obrazovky |
| Dlouhý stisk (2s) | Reset aktuálního měření |
| Dvojklik | Zapnutí/vypnutí displeje (úspora baterie) |
| Přidržení při startu | Tovární reset (smazání NVS) |
